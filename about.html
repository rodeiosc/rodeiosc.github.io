Visão geral do projeto

A estação utiliza uma Ambient Weather WS-2902 para captar temperatura, vento, umidade, chuva e outros parâmetros meteorológicos. Os dados são enviados por rede e aproveitados localmente para alimentar este site.

Em paralelo, um Raspberry Pi 3 Model B atua como servidor local para processar, organizar e apresentar as informações, gerar gráficos, automatizar rotinas e integrar a captura de imagens da câmera HILOOK IPC-B121H-L.

O ambiente também conta com armazenamento dedicado em um case externo Exbom 3,5 SATA USB 3.0 para HD/SSD, oferecendo praticidade para expansão e organização dos arquivos gerados.

Fluxo técnico

1. Coleta local A estação registra variáveis atmosféricas e envia leituras para o ambiente local.
2. Processamento O Raspberry organiza dados, converte medidas e prepara arquivos estáticos.
3. Captura visual A câmera registra imagens periódicas para acompanhamento do tempo.
4. Publicação Os arquivos HTML, JSON, CSV e mídia são enviados ao GitHub Pages.

Equipamentos principais

Componentes que sustentam a operação da estação e da plataforma local.

Ambient Weather WS-2902

Responsável pela captura dos dados meteorológicos locais, incluindo temperatura, vento, umidade e chuva.

Raspberry Pi 3 Model B

Servidor local que executa processamento, automações, integração da câmera e preparação dos arquivos publicados.

HILOOK IPC-B121H-L

Câmera IP bullet utilizada para captura de imagens da área monitorada e acompanhamento visual das condições do tempo.

Case Externo Exbom 3,5 SATA USB 3.0

Usado com HD/SSD para hospedagem local, armazenamento, processamento de fotos e organização do conteúdo gerado.
